# Credits & sources

RoboVoice stands on a lot of community work. Thank you 🙏

## Voice packs

The bundled character packs were created by the robot‑vacuum community and shared on
**[vacuum.mindsolo.net](https://vacuum.mindsolo.net)**, **[4PDA](https://4pda.to/)**, VK groups and
GitHub. RoboVoice only **re‑maps** them to the `trouver.vacuum.r2567r` event layout — the voice
acting and editing are the original authors'. Known contributors include **Rubler4ik** (Кожаные
ублюдки) and the many anonymous 4PDA/VK pack makers behind Кузя, Советские фильмы, Дерзкая Галя,
Элеонора, Добкин и др.

If you made one of these packs and want credit added — or want it **removed** — please open an
issue and we'll act on it immediately.
